# About
Simple ftp-style client.

# Installation
To install, simply use `pip` or `easy_install`:

```bash
$ pip install --upgrade gdriveshell
```
or
```bash
$ easy_install --upgrade gdriveshell
```

# Python Version
Developed using 3.5.1
